# جعبه‌ابزارهای کاربردی من

مجموعه‌ای از ابزارهای تحت وب که **بدون نصب** و **کاملاً در مرورگر** اجرا می‌شوند.  
هدف: حل مشکلات کوچک روزمره با سرعت و حفظ حریم خصوصی.

##  چه مشکلی را حل می‌کند؟
- دیگر نیازی به نصب نرم‌افزارهای سنگین برای کارهای ساده نیست.
- ابزارها **آفلاین** و **محرمانه** هستند – داده‌ها به جایی ارسال نمی‌شوند.
- مناسب برای کاربران غیرفنی، دانشجویان، طراحان و هر کسی که به یک ابزار سریع نیاز دارد.
